<h3><a href="https://codeforces.com/contest/2166/problem/E" target="_blank" rel="noopener noreferrer">Binary Wine</a></h3>

<div class="header"><div class="title">E. Binary Wine</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>  </p><p>You are given $$$n$$$ integers $$$a_1,a_2,\ldots,a_n$$$ within the range $$$[0,2^{30})$$$.</p><p>You can spend $$$1$$$ coin to increase any $$$a_i$$$ by $$$1$$$. You can perform this operation any number of times.</p><p>You need to solve $$$q$$$ queries; for each query, you are given an integer $$$c$$$, also in the range $$$[0,2^{30})$$$. You would like it if there exists a sequence $$$b$$$ of length $$$n$$$ with the following properties: </p><ul> <li> For every $$$1\le i\le n$$$, $$$0\le b_i\le a_i$$$. </li><li> $$$b_1\oplus b_2\oplus\ldots\oplus b_n=c$$$, where $$$\oplus$$$ denotes the <a href="https://en.wikipedia.org/wiki/Bitwise_operation#XOR">bitwise XOR operation</a>.  </li></ul><p>Please calculate the minimum number of coins you will have to spend, such that there exists a suitable $$$b$$$.</p><p>The queries are <span class="tex-font-style-bf">independent</span>, meaning that any operations you perform on the sequence $$$a$$$ will not impact future queries.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case consists of two integers $$$n,q$$$ ($$$1\le n\le5\cdot10^5$$$, $$$1\le q\le5\cdot10^4$$$) — the length of sequence $$$a$$$ and the number of queries.</p><p>The second line of each test case contains $$$n$$$ integers $$$a_1,a_2,\ldots,a_n$$$ ($$$0\le a_i<2^{30}$$$) — the initial sequence $$$a$$$.</p><p>Each of the next $$$q$$$ lines contains a single integer $$$c$$$ ($$$0\le c<2^{30}$$$) — the target XOR.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$5\cdot10^5$$$. </p><p>It is guaranteed that the sum of $$$q$$$ over all test cases does not exceed $$$5\cdot10^4$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each
