Pizza Delivery</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Courier YF has earned the title of the best pizza delivery person GR. The manager does not like him, so he decided to give him a very difficult task. The manager provided him with $$$n$$$ coordinates of houses $$$(x_i, y_i)$$$ where he needs to deliver the pizza. He will deliver the pizza in the following way:</p><ul> <li> GR pizza is prepared at the point ($$$Ax, Ay$$$) and YF starts the delivery from this point. </li><li> To deliver the pizza, he can move from the point ($$$x, y$$$) to <span class="tex-font-style-bf">three</span> points ($$$x+1, y$$$), ($$$x, y+1$$$), ($$$x, y-1$$$). </li><li> After he has delivered all the pizzas, he returns home to the point ($$$Bx, By$$$). </li></ul> <p>Each move takes him exactly one second, and handing over the pizza to the customer takes $$$0$$$ seconds. The manager wants the delivery to be as fast as possible. You need to find the minimum delivery time for all GR pizzas. It is guaranteed that delivery is always possible.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of several test cases. Ay, Bx, By \le 10^9$$$) — the number of houses for delivery, as well as the coordinates of the start and end points.</p><p>The second line of each test case contains $$$n$$$ integers $$$x_1, x_2, \dots, x_n$$$ ($$$Ax \lt x_i \lt Bx$$$).</p><p>The third line of each test case contains $$$n$$$ integers $$$y_1, y_2, \dots, y_n$$$ ($$$1 \le y_i \le 10^9$$$).</p><p>It is guaranteed that the sum of the values of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single
