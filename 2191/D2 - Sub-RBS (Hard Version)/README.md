<h3><a href="https://codeforces.com/contest/2191/problem/D2" target="_blank" rel="noopener noreferrer">Sub-RBS (Hard Version)</a></h3>

<div class="header"><div class="title">D2. Sub-RBS (Hard Version)</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p><span class="tex-font-style-bf">This is the hard version of the problem. The difference between the versions is that in this version, you need to find the sum of scores over all subsequences of $$$s$$$; $$$s$$$ is not necessarily a <span class="tex-font-style-it">regular bracket sequence</span>, and the constraints on $$$n$$$ are lower</span>.</p><p>We say that a bracket sequence $$$a$$$ is <span class="tex-font-style-bf">better</span> than a bracket sequence $$$b$$$ if one of the following holds: </p><ul> <li> $$$b$$$ is a prefix of $$$a$$$, but $$$a \ne b$$$; or </li><li> let $$$i$$$ be the first position (if it exists) where $$$a_i \neq b_i$$$, then $$$\color{red}{a_i = \texttt{(}}$$$ and $$$\color{red}{b_i = \texttt{)}}$$$. </li></ul><p>For an arbitrary bracket sequence $$$t$$$, we define its <span class="tex-font-style-bf">score</span> in the following way:</p><ul> <li> If $$$t$$$ is <span class="tex-font-style-bf">not</span> a <span class="tex-font-style-it">regular bracket sequence</span>$$$^{\text{∗}}$$$, the score is $$$0$$$. </li><li> If there exists a regular bracket subsequence $$$^{\text{†}}$$$ $$$r$$$ of $$$t$$$ such that $$$r$$$ is <span class="tex-font-style-bf">better</span> than $$$t$$$, then the score is equal to the maximum value of $$$|r|$$$ over all such subsequences $$$r$$$. </li><li> Otherwise, the score is $$$0$$$. </li></ul><p>In other words, the score of $$$t$$$ is the length of the longest regular bracket subsequence of $$$t$$$ which is better than $$$t$$$. If $$$t$$$ is not a regular bracket sequence, or if no regular subsequence better than $$$t$$$ exists, the score is $$$0$$$.</p><p>You are given a bracket sequence $$$s$$$ of length $$$n$$$. Find the sum of the scores of all non-empty subsequences of $$$s$$$ modulo $$$998\,244\,353$$$.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A <span class="tex-font-style-it">regular bracket sequence</span> is a bracket sequence that can be transformed into a correct arithmetic expression by inserting the characters $$$\texttt{1}$$$ and $$$\texttt{+}$$$ between the original characters of the sequence. For example:</p><ul> <li> bracket sequences $$$\texttt{()()}$$$ and $$$\texttt{(())}$$$ are regular (the resulting expressions are $$$\texttt{(1)+(1)}$$$ and $$$\texttt{((1+1)+1)}$$$); </li><li> bracket sequences $$$\texttt{)(}$$$, $$$\texttt{(}$$$, and $$$\texttt{)}$$$ are not. </li></ul><p>$$$^{\text{†}}$$$A sequence $$$a$$$ is a subsequence of a sequence $$$b$$$ if $$$a$$$ can be obtained from $$$b$$$ by the deletion of several (possibly, zero or all) element from arbitrary positions. </p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 30$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 100$$$) — the length of the string $$$s$$$.</p><p>The second line of each test case contains a sequence $$$s$$$ of length $$$n$$$ consisting only of characters $$$\texttt{(}$$$ and $$$\texttt{)}$$$.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$100$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, print a single integer — the sum of the scores of all subsequences
