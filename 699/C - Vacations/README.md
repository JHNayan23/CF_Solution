<div class="header"><div class="title">C. So he decided to improve his IT skills and do sport. Vasya knows the following information about each of this <span class="tex-span"><i>n</i></span> days: whether that gym opened and whether a contest was carried out in the Internet on that day. For the <span class="tex-span"><i>i</i></span>-th day there are four options:</p><ol> <li> on this day the gym is closed and the contest is not carried out; </li><li> on this day the gym is closed and the contest is carried out; </li><li> on this day the gym is open and the contest is not carried out; </li><li> on this day the gym is open and the contest is carried out. </li></ol><p>On each of days Vasya can either have a rest or write the contest (if it is carried out on this day), or do sport (if the gym is open on this day).</p><p>Find the minimum number of days on which Vasya will have a rest (it means, he will not do sport and write the contest at the same time).
