<div class="header"><div class="title">E. Hidden Knowledge of the Ancients</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>In the world of <span class="tex-font-style-it">Deepwoken</span>, there exists an ancient artifact — <span class="tex-font-style-it">Tablet of Infinite Knowledge</span>, on which a sequence of $$$n$$$ mysterious symbols (each symbol is an integer) is engraved.</p><p>It is said that the true power of the artifact can only be revealed by finding all <span class="tex-font-style-it">sacred fragments</span> — continuous segments of the tablet that contain exactly $$$k$$$ distinct numbers, and their length must be between $$$l$$$ and $$$r$$$ (inclusive).</p><p>Formally: Given a sequence $$$a$$$ of length $$$n$$$ and integers $$$k$$$, $$$l$$$, $$$r$$$. You need to find the number of such boundaries $$$b$$$ and $$$c$$$ such that: </p><ul> <li> $$$1 \le b \le c \le n$$$; </li><li> among the elements [$$$a_{b}, a_{b + 1}, \dots, a_{c}$$$] there are exactly $$$k$$$ distinct numbers; </li><li> $$$l \leq c - b + 1 \leq r$$$. </li></ul></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of several test 10^9)$$$ — the mysterious symbols.</p><p>It is guaranteed that the total value of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test, output a single integer on a separate line — the number of continuous subarrays that meet the specified conditions.</p></div><div
